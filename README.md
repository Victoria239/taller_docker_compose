# Taller Docker Compose - API de Tareas

Aplicación pequeña desarrollada con FastAPI y PostgreSQL para el taller de Docker Compose.

La aplicación permite:

- Crear tareas.
- Listar tareas.
- Eliminar tareas.
- Consultar tareas por ID.

Tecnologías utilizadas:

- Python
- FastAPI
- PostgreSQL
- Docker
- Docker Compose
- Pytest
